A Deferred Correction, Continuous Galerkin Method for Curvilinear Staggered-Grid Lagrangian Hydrodynamics

Track CFD 5

Presenting speaker Steven Walton (LANL)

Authors Steven Walton (LANL); Svetlana Tokareva (LANL); Nathaniel Morgan (LANL)

Abstract

We present a staggered-grid hydrodynamics scheme which employs a Deferred Correction time integration strategy coupled with a continuous Galerkin discretization in space (cG-DeC). The momentum equation is solved using a continuous finite element spatial discretization while the thermodynamic variables are allowed to be discontinuous. For both the kinematic and thermodynamic variables, the solve is a simple scalar divide.
