A New Family of Explicit Runge–Kutta Integrators for Continuous Finite Element Discretizations

Track Time integration 1

Presenting speaker Tommaso Buvoli (Tulane)

Authors Tommaso Buvoli (Tulane), Steven Walton (LANL)

Abstract

Explicit Runge–Kutta (ERK) methods are commonly used to integrate non‑stiff, advection‑dominated problems discretized with continuous finite elements. However, their performance is often limited by the cost of repeatedly solving linear systems involving a non‑diagonal mass matrix. Although mass lumping can eliminate this cost, it also introduces errors that destroy temporal convergence.This talk introduces a new class of ERK methods that significantly reduce the cost of these linear solves while retaining temporal convergence. The key idea is to reuse stage‑derivative information from previous timesteps to generate highly accurate initial guesses for the linear solver. This approach reduces computational cost by requiring fewer solver iterations without sacrificing temporal accuracy. The resulting methodology provides an efficient, easily implementable, and more accurate alternative to traditional mass‑lumping techniques.
