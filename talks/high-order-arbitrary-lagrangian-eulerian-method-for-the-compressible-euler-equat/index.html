High-order Arbitrary Lagrangian Eulerian method for the compressible Euler equations

Track CFD 6

Presenting speaker Jean-Luc Guermond (Texas A&M)

Authors Jean-Luc Guermond (Texas A&M), Laura Saavedra (University of Santiago de Compostello)

Abstract

In this work we generalize an idea introduced in Guermond-Popov-Saavedra-Yong (2017) for solving the compressible Euler equations using an arbitrary Lagrangian Eulerian method (ALE). It is shown therein that the discrete geometric conservation law is satisfied to any order in time by augmenting the system with the Reynolds transport equation to define the finite element mass matrix. The first-order variant of the method is shown to be invariant domain preserving (IDP). Using limiting techniques based on theflux corrected transport method by (Boris-Book (1973), Zalesak (1979) (FCT) and variation thereof based on convex limiting (Zhang-Shu (2011), Guermond-Nazarov-Popov-Tomas (2018), the method can be made higher-order accurate in time and space and IDP.Recall that in the FCT paradigm (and generalizations thereof) a \`\`non-admissible'' but accurate high-order solution is squeezed in the direction of a \`\`good'' low-order solution, while maintaining conservation, until the squeezed high-order solution is in some targeted admissible set. The main problem with these techniques is that they require to have at hand an IDP low order method that carries exactly the same mass (on average) as the high-order one. While the requirement of having two methods carrying the same mass is not a severe constraint when low-order approximations, see (Abgrall etal (2017), Abgrall-Bacigaluppi-Tokareva (2019)), it becomes cumbersome when using high-order elements, and it is even a dead end when one wants to use spectral methods (eg. Fourier approximation).In the light of the argumentation above, and in the continuation of Guermond-Wang (2025) we propose a new point of view that consists of completely abandoning Zalesak's paradigm. It is the objective of the present work to show that this idea can be extended to the ALE approximation of the compressible Euler equations using high polynomial degrees.
