An Efficient and Robust Ghost Fluid Method for Arbitrary Number of Materials

Track CFD 5

Presenting speaker Tariq D. Aslam (Los Alamos National Laboratory)

Authors Tariq D. Aslam (Los Alamos National Laboratory); Eduardo Lozano (Los Alamos National Laboratory)

Abstract

The original Ghost Fluid Method (GFM) was devised to track two different fluids on an Eulerian grid via the use of a single level set function and storage of the solution state for both fluids at every computational point. It allows for distinct equations of state (EOS) for each fluid and discrete jumps in density and transverse velocity. A new efficient level set methodology, where only a single integer and single real variable need to be stored at each computational point will be presented. A variety of researchers have improved the GFM in terms of robustly treating interfaces with vastly different material properties, for example condensed phase materials near gases. Furthermore, recent advances in provably robust numerical methods for compressible flow, Invariant Domain Preserving (IDP) methods, have been extended to condensed phase EOS. The work to be presented will include incorporation of IDP strategies for robust treatment of the conservation laws and extensions to the ghost region states of GFM for stable computations.
